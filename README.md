# LibraryManagementSystemOS

#Features

Multi-threaded TCP server using sockets
Client menu interface for easier interaction
Register and login users
Role-based access (Student / Librarian)
Create and manage library records
Assign and update record status
Persistent storage using text files
Console logging with timestamps
Thread-safe data handling

#Requirements

Java 8 or higher
Eclipse or any Java IDE
No external libraries required

#How to Run

Open the project in Eclipse.
Run the Server.java file to start the server.
Run Client.java in a separate console window.

Use the menu to register, login, and manage records.

You may run multiple clients simultaneously to test concurrency.
